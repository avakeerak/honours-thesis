# Mapping the Recent Evolution of Glacial Lakes in Coastal British Columbia
*Author:* Ava Keerak

This repository contains the R code used for my honours thesis titled "Mapping the Recent Evolution of Glacial Lakes in Coastal British Columbia". R was used instead
of Python becasue I wanted to challenge myself to use a language that I was less familiar with.

All of the files are named after their corresponding section in my honours thesis. For example, the R code that was used to create the figures included in section "3.2  Total Changes in Glacial Lake Area" is named "total_changes_lake_area_3.2.R".

Most files have "your_data_here" included in the code in place of the dataframe that I used. The data was downloaded from ArcGIS and cleaned in Microsoft Excel (version 2605) unless otherwise indicated.

R version 4.3.2 was used, as well as the CRAN libraries "readxl", "writexl", "dplyr", "extrafont", and "ggplot2". To calculate the rates of change, I used Microsoft Excel code created by Brandon Karchewski for the Fall 2025 course "GOPH 419 - Computational Methods for Geophysicists".
